# RRWeb React Kit 用户手册

## 1. 产品简介
RRWeb React Kit 是一个帮助开发者在 React 应用中快速集成“网页回放”功能的工具。类似于 Hotjar 或 FullStory 的基础版。

## 2. 快速使用步骤
### 第一步：开启录制
点击页面上的“开始录制”红色按钮。此时你会看到状态指示灯变为“播放中...”。

### 第二步：模拟操作
在下方的“交互测试区”进行以下尝试：
- 在输入框输入一段话。
- 点击按钮。
- 将鼠标悬停在紫色色块上。

### 第三步：停止与保存
完成操作后，点击“停止录制”。此时下方会自动出现“回放区域”。

### 第四步：观看回放
- 点击“播放”按钮观察你之前的操作。
- 使用下拉框尝试 4.0x 倍速观看。
- 进度条会实时显示当前回放的位置。

## 3. 常见问题 (FAQ)
**Q: 为什么我回放时看不到我刚才的弹窗？**
A: 请确保弹窗是在点击“开始录制”之后触发的，rrweb 只能捕捉开启录制后的 DOM 变动。

**Q: 录制的内容会上传到服务器吗？**
A: 本演示版本所有数据均保存在浏览器内存中，刷新页面后会丢失。如需保存，你需要自行实现事件上报接口。
