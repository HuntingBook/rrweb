# RRWeb React Kit 界面与逻辑设计文档

## 1. 设计初衷
为了提供一个既美观又实用的演示工具，本项目在界面设计上追求“专业感”和“易用性”。

## 2. 界面设计 (UI Design)
### 2.1 布局结构
- **TopBar**: 顶部导航栏，采用毛玻璃效果 (Glassmorphism)，固定在页面顶部。
- **MainContent**: 居中容器，限制最大宽度 (1200px)，保证在大屏幕上的视觉体验。
- **Card**: 所有功能区块封装在卡片容器中，具有柔和的投影 (Shadow) 和圆角 (Radius)。
- **Footer**: 底部信息栏，保持简洁。

### 2.2 配色方案
- **主色 (Primary)**: 靛蓝色 (#6366f1)，用于开始录制、播放等主操作。
- **危险色 (Danger)**: 红色 (#ef4444)，用于停止录制，提供强烈的视觉警示。
- **背景色**: 灰白色 (#f8fafc)，辅助色使用深灰色和浅灰色。

### 2.3 交互动效
- **录制状态指示**: 录制时状态灯带有呼吸呼吸灯动画效果 (`pulse`)。
- **悬停效果**: 按钮和可交互元素在悬停时会有位移或阴影加深效果。

## 3. 逻辑设计 (Logic Design)
### 3.1 性能优化
- **渲染隔离**: 针对播放器每秒 60 帧的高频进度更新，采用了组件拆分和 `React.memo` 优化。
- **ControlBar 隔离**: 将控制按钮（播放/倍速）从进度文本中分离，防止进度更新导致下拉框或按钮由于父级重绘而产生交互卡顿。

### 3.2 响应式设计
- **播放器容器**: 采用 `overflow: auto` 处理，确保当录制的内容分辨率超过当前容器时，用户依然可以通过滚动条查看到所有区域。
- **左侧对齐**: 回放内容强制左上角对齐 (`textAlign: left`)，确保比例还原的一致性。
